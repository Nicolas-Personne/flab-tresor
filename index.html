LE JEU

  • Règle n°1 : Ne partagez aucun indice trouvé.
  • Règle n°2 : Vous ne pouvez avancer qu'en résolvant les énigmes dans l'ordre.
  • Règle n°3 : L'entraide est permise… jusqu'au moment où le trésor est découvert.
  • Règle n°4 : Amusez-vous et profitez de l'aventure !

Bienvenue dans le jeu de piste du mariage

Un trésor est dissimulé quelque part dans ce lieu enchanteur.

Pour le découvrir, vous devrez résoudre des énigmes et retrouver des codes disséminés un peu partout.

Chaque code ne pourra être débloqué qu'en trouvant un mot-clé ou une réponse, déduite de l'indice précédent.

Attention… Les joueurs qui mettront la main sur le trésor seront liés par une magie ancienne, les empêchant d'aider les autres participants.
